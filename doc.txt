1) МАКРОСЫ
Синтаксис:
    #def @имя_макроса(входные_параметры,...){тело_макроса};


#def @SUM(a,b){a + b};  // обьявление макрса

#def   @SUM(a,b){a+b}; // другой формат обьявления макроса

#def   @SUM(a,b)  {a+b}; // другой формат обьявления макроса

#def   @SUM(a,b)  {  // данное обьявление верно но включаються также и \n
a+b
};

#def @SUM(a, b){a+b};  // обьявление входных параметров с пробеламим выдаст ошибку, т.к. пробелы тоже учитываються
